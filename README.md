Exercice 1 : 

Cet exercice consiste à développer un programme Java permettant de gérer les notes d’un groupe d’étudiants à l’aide d’un tableau. Le programme lit les notes saisies au clavier, les stocke dans un tableau, puis permet d’effectuer plusieurs traitements : trier les notes, calculer la moyenne, déterminer la note minimale et maximale, et compter le nombre d’étudiants ayant une note donnée. L’objectif est de maîtriser la manipulation des tableaux, les boucles, et l’utilisation de la classe Arrays.

Exercice 2 : 

L’objectif de cet exercice est de développer un programme Java qui lit un verbe du premier groupe  et affiche sa conjugaison au présent de l’indicatif. Le programme vérifie que le verbe se termine bien par « er » et suppose qu’il s’agit d’un verbe régulier. Cet exercice permet de pratiquer la manipulation des chaînes de caractères et les opérations sur les sous-chaînes.

Exercice 4 : 

Cet exercice consiste à écrire un programme Java qui lit une chaîne de caractères saisie au clavier et compte le nombre d’occurrences de chaque lettre de l’alphabet, sans distinguer entre les majuscules et les minuscules. Les résultats sont stockés dans un tableau de 26 cases, correspondant aux lettres de « a » à « z ». Le programme affiche uniquement les lettres apparaissant au moins une fois dans le texte. Cet exercice permet de travailler sur les tableaux, les caractères, et la gestion des chaînes de caractères.
